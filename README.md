# Nacionale Streetwear

Nacionale je Next.js sajt sa 3D animacijama, fokusiran na premium streetwear prezentaciju i moderni vizuelni identitet.

## Tehnologije

- Next.js (App Router)
- Tailwind CSS
- GSAP i Three.js
- TypeScript

## Lokalni razvoj

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
